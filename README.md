# docker-komodo


## Clone the repo (or fork it)
```
git clone https://github.com/Monaize/docker-insight.git

cd docker-insight.git
``` 

## Edit .env file with your assetchain parameters
* Defaults: 
    * AC_NAME=MNZ
    * AC_SUPPLY=257142858

## Get up !
```
docker-compose up
```
